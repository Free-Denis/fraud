## 1. Бизнес-контекст
Банк несет прямые убытки по банковским гарантиям, когда подрядчики срывают государственные контракты. Часто такое допускают фирмы-однодневки, которые изначально не имеют ресурсов для выполнения работы, но выигрывают тендеры.

Гипотеза: Существует паттерн недобросовестных поставщиков, связанный с вероятность срыва госконтракта и попадания компании в Реестр недобросовестных поставщиков. 
И возрастает при сочетании факторов:
Приближении уставного капитала к минимально допустимому;
Меньшем количестве сотрудников;
Меньшем возрасте фирмы.
